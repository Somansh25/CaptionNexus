How does the captioning engine interpret images?

The system uses the Salesforce BLIP (Bootstrapping Language-Image Pre-training) model, which employs a vision transformer backbone and a multimodal text decoder for state-of-the-art accuracy.

What file formats and sizes are supported?

We support standard binary formats including PNG, JPG, JPEG, and WEBP. The maximum permissible file size is currently capped at 16MB per ingestion cycle.

Is the history data persistent or volatile?

If the datastore cluster is ONLINE, your history is persisted in a MongoDB database. If the cluster is OFFLINE, the system utilizes a volatile RAM fallback.
